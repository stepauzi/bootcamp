# Data Structure
## ArrayList (List)
- with order
- allow duplicated object
- get(index)
- remove(Object)
- remove(index)
- addLast(Object)

## LinkedList (Deque/ Queue/ List)
- with order
- allow duplicated object
- remove head/tail (java LinkedList -> double LinkedList)
- remove(Object)
- addLast(Object)

## ArrayDeque (Deque/ Queue)
- with order
- allow duplicated object

## HashSet (Set)
- No order
- disallow duplicated object
- Underlying data structure (array? linkedlist?)

## HashMap (Map)
- No order
- Entry (Key + Value)
- Underlying data structure (array? linkedlist?)
